# Zuu

A git hook test runner for rust programer.

## Installation

```sh
cargo install zuu
```

## Requirement

Wget and git must be installed on your system

## Track repository

```bash
git init
zuu init
```

## Usage

```sh
git add .
git commit -m "commit msg"
```

## Manual usage

```bash
zuu
zuu --watch
zuu --watch 5 
```

Time is in seconds

## Upgrade the hook file

```bash
zuu upgrade
```

## Display help

```bash
zuu --help
```
