# Zuu 

A git hook test runner for rust programer.

## Installation

```sh
cargo install zuu
```

## Arch user 

```bash
yay -S zuu
```

## Track repository

```bash
git init
```

## Usage

```sh
git add .
git commit -m "commit msg"
```
